# Week 1: Introduction and Mindset Shift

## Day 1: Monday - Onboarding and MGSWeb3 Origin Story

Objectives:
- Align mindset and define expectations
- Understand why MGSWeb3 exists
- Learn accountability systems
- Understand the cohort journey: generalization to specialization to projects to showcase
- Learn about GitHub deliverables and public learning
- Understand community structure
- Learn about the eviction system at Week 2 and Week 4 checkpoints

Actions:
- Confirm readiness to participate
- Create a GitHub account if you do not have one

Assignment:
- Create a file named: your-name-onboarding.md
- Document what you learned from the onboarding session
- Include your key takeaways about MGSWeb3 origin story
- Note your understanding of the cohort journey and accountability systems

Deadline: Submit by the deadline specified by your tutor. Assignments given on Monday are due at the tutor deadline.

## Day 2: Wednesday - Intro to Web3: Blockchain and Internet Evolution

Objectives:
- Understand why Web3 exists by tracing how Bitcoin and Ethereum changed ownership, money, and creative freedom
- Learn Web1 to Web2 to Web3 evolution
- Understand freedom vs platforms vs ownership
- Learn what blockchain is: blocks, nodes, immutability
- Understand Web2 pain points: censorship, middlemen, exploitation
- Learn how Bitcoin introduced digital scarcity
- Learn how Ethereum added programmability and creativity
- Understand smart contract concept at surface level
- Learn Web3 opportunities for creatives: ownership, royalties, collaboration
- Get surface level preview of wallets, dApps, tokens

Assignment:
- Create a file named: your-name-blockchain-intro.md
- Document your understanding of Web1 to Web2 to Web3 evolution
- Explain why blockchain exists and how it differs from traditional systems
- Note your key takeaways about Bitcoin and Ethereum
- Include your thoughts on Web3 opportunities for creatives

Deadline: Submit by the deadline specified by your tutor.

## Day 3: Friday - GitHub for Creatives Part 1: Getting Started

Objectives:
- Eliminate fear of GitHub
- Understand GitHub as creative proof of work and reputation layer
- Learn what version control is and why it matters for creatives
- Learn what GitHub is: basics of repo, commit, pull request
- Understand creative use cases: portfolios, collaborations, idea banks

Hands-on activities:
- Create a GitHub repo titled "My Web3 Journey"
- Write a short intro in README.md
- Save work and push first commit

Assignment:
- Create a file named: your-name-github-setup.md
- Document your experience creating your GitHub repo
- Include screenshots of your first commit
- Write your Web3 story in the README.md of your personal repo
- Add a link to your "My Web3 Journey" repo in your assignment file

Deadline: Submit by the deadline specified by your tutor.

## Deliverable

Submit all three day assignments by the deadlines specified by your tutor.

Your Web3 Story in your personal repo includes:
- Why you joined MGSWeb3
- What you hope to learn
- Your creative background or interests
- Your initial thoughts about Web3

## How to Submit Your Assignment

Follow these steps:

1. Fork this repository to your GitHub account
2. Create a GitHub repo titled "My Web3 Journey" for your personal learning journey
3. Complete all three day assignments:
   - Day 1 Monday: your-name-onboarding.md
   - Day 2 Wednesday: your-name-blockchain-intro.md
   - Day 3 Friday: your-name-github-setup.md
4. Create a new branch for your submission:
   git checkout -b week-1-assignment

5. Add your work inside the submissions folder with day folders:
   submissions/
       └── week-1/
           └── your-name/
               ├── day-1/
               │   └── your-name-onboarding.md
               ├── day-2/
               │   └── your-name-blockchain-intro.md
               └── day-3/
                   └── your-name-github-setup.md

6. In your day-3 assignment file, include a link to your "My Web3 Journey" repo

7. Commit and push your changes:
   git add .
   git commit -m "chore: submit week 1 assignment"
   git push origin week-1-assignment

8. Create a Pull Request to the main repository. Your PR includes:
   - A brief summary of what you learned each day
   - A link to your "My Web3 Journey" repo

Important: Replace "your-name" with your actual name in all file names and folder paths.

## Evaluation Criteria

Submissions are evaluated based on:
- Completeness of your Web3 story
- Clarity of documentation
- Consistency in following submission structure
- Active GitHub repo with first commit
