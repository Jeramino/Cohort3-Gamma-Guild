# MGSWeb3 Cohort 3 Generalization Phase

## Overview

This repository holds resources, assignments, and contributions for the 4-week Generalization Phase of MGSWeb3 Cohort 3. You complete tasks, document your process, and submit work via PR to this repository.

Goal: Move from Web3 Beginner to Web3 Collaborative Contributor

Outputs: Active GitHub usage, conceptual fluency, project readiness

Eviction Checkpoints: End of Week 2 and Week 4

The Generalization Phase introduces Web3 fundamentals, blockchain technology, smart contracts, DAOs, and tools for active contribution in Web3.

You learn through hands-on sessions, guided mentorship, and community collaboration.

You learn:
- How Web1 became Web2 and Web2 became Web3
- Why blockchain exists
- How blockchain, smart contracts, and decentralized governance work
- How to use GitHub as proof of work and reputation layer
- How to use Web3 tools: wallets, dApps, security practices
- How tokenomics, Web3 culture, and creative revenue streams work
- How to build public narrative and prepare for specialization

## What You Find Here

This repository coordinates the program. It includes:

- Assignments: Weekly exercises and challenges
- Resources: Reading materials, slides, and reference documentation
- Submissions: Your work submitted via Pull Requests
- Reflections: Weekly learning reflections and deliverables

## Program Structure

### Week 1: Introduction and Mindset Shift

Monday: Onboarding and MGSWeb3 Origin Story

What to do on Monday:
- Attend the onboarding session
- Learn about MGSWeb3 origin story
- Understand the cohort journey: generalization to specialization to projects to showcase
- Learn about GitHub deliverables and public learning
- Understand community structure
- Learn about the eviction system at Week 2 and Week 4 checkpoints
- Confirm readiness to participate
- Create a GitHub account if you do not have one

How to submit for Week 1:
1. Fork this repository to your GitHub account
2. Create a GitHub repo titled "My Web3 Journey" for your personal learning journey
3. Complete all three day assignments:
   - Day 1 Monday: Create your-name-onboarding.md
   - Day 2 Wednesday: Create your-name-blockchain-intro.md
   - Day 3 Friday: Create your-name-github-setup.md
4. Submit each assignment by the deadline specified by your tutor
5. Create a new branch for your submission: git checkout -b week-1-assignment
6. Add your work inside the submissions folder with day folders:
   submissions/week-1/your-name/day-1/your-name-onboarding.md
   submissions/week-1/your-name/day-2/your-name-blockchain-intro.md
   submissions/week-1/your-name/day-3/your-name-github-setup.md
7. In your day-3 assignment file, include a link to your "My Web3 Journey" repo
8. Commit and push your changes: git add . then git commit -m "chore: submit week 1 assignment" then git push origin week-1-assignment
9. Create a Pull Request to the main repository with your name in the PR title

Important: Replace "your-name" with your actual name in all file names and folder paths. For example: submissions/week-1/john-doe/day-1/john-doe-onboarding.md

Deadline: Submit each assignment by the deadline specified by your tutor. Assignments given on Monday must be submitted by the tutor deadline.

Wednesday: Intro to Web3: Blockchain and Internet Evolution from Bitcoin to Ethereum to Creative Internet

Friday: GitHub for Creatives Part 1: Getting Started

Deliverable: Submit all three day assignments by the deadlines specified by your tutor. Each assignment file should be named with your name plus the assignment name.

### Week 2: Blockchain Engine, Smart Contracts and Governance

Monday: Blockchain and Web3 Fundamentals Deep Dive

Wednesday: Smart Contracts 101: The Trustless Engine

Friday: DAOs and Discord Management Guest Session

Deliverables:
- Explain a transaction on Etherscan in your GitHub repo
- Explain a real smart contract case study in README.md
- Participate in DAO simulation

Eviction Checkpoint: You need an active GitHub repo with at least 2 pushed submissions, at least 1 reflection post on social media, and participation in DAO simulation.

### Week 3: Tooling, Culture and Financial Literacy

Monday: Web3 Wallets, dApps and Security for Creatives

Wednesday: Tokenomics for Creatives plus Web3 Culture: Degens and Narratives

Friday: Web3 Marketplaces and Creative Revenue Streams

Deliverables:
- Connect to Sepolia testnet and simulate transaction
- Create a sample token concept in groups and pitch with a meme
- List 3 Web3 earning paths for your creative type

### Week 4: Ethics, AI Power and Public Narrative

Monday: Web3 Law and IP for Creatives Guest Session

Wednesday: AI for Creatives in Web3 Guest Session

Friday: Building in Public: Crafting Your Web3 Narrative

Deliverables:
- Add an IP disclaimer to your GitHub repo
- Create an AI-generated creative asset supporting your Web3 roadmap
- Publish "My future in Web3" roadmap via GitHub and post

Eviction Checkpoint: You complete all required deliverables and demonstrate active participation.

## How to Participate

- Fork this repository to your GitHub account
- Create a GitHub repo titled "My Web3 Journey" for your personal learning journey
- Follow weekly instructions posted in /assignments
- Document your work with notes, screenshots, or reflections
- Submit your work via Pull Request to this repository
- Each PR includes:
  - Your folder under /submissions/your-name/week-X/ with day-1, day-2, day-3 folders for Week 1
  - Assignment files named with your name plus assignment name
  - A summary of what you learned
  - Supporting files: notes, screenshots, reflections, or code
- Submit assignments by the deadline specified by your tutor
- Post reflections on social media at least 1 per week from Week 2 onwards
